ゆゆbot の使い方と機能を超ざっくり解説
このグループにいる「ゆゆbot」は、みんなのお手伝いをしてくれる便利なやつ。
使い方はちょーかんたん、メッセージでコマンドを打つだけ。
（ちょっとだけ返信が遅れることもあるから、気長に待っててね！）
​
​■ 使えるコマンドリスト
​/today
→ 今日の日付と、登録してあるイベントを教えてくれる
​おみくじ
→ 今日の運勢を占ってくれる。
管理者の人には、特別な大吉が出やすいらしい
​/day-write yyyy-mm-dd [イベント名]
→ 特定の日付にイベントを登録できる
例：/day-write 2025-07-03 だれかさんの誕生日
​/day-view
→ 登録したイベントのリストを見せてくれる
​/yes-or-no
→ 「Yes」か「No」をランダムで返してくれる
どうしようか迷ったときに！
​/wiki/検索キーワード
→ Wikipediaで検索して、ざっくりとした説明を表示してくれる
例：/wiki/天理教
​/scratch-user/ユーザー名
→ 指定したScratchユーザーの情報を見せてくれる
​/scratch-project/プロジェクトID
→ 指定したScratchプロジェクトの情報を見せてくれる
​/member
→ グループのメンバー一覧を表示してくれるよ
​/member-name
→ プロフアイコンなしのメンバー一覧を表示してくれるから、ルーレットとかに便利
​ゆゆゆ
→ ゆゆゆを強制的に呼び出すよ
​はんせい
→ はんせいを強制的に呼び出すよ
​からめり
→ からめりを強制的に呼び出すよ
いろいろあぷり
→リンクくれるよ
喘げ
→botが喘ぐよ
おやすみ
→返事してくれるよ
​
​■ 勝手に動いてくれるしゅごい機能
​毎日0時になったら、今日の日付とイベントを自動で投稿してくれる
​グループメンバーのリストを、スプレッドシートに自動で保存・更新してくれる
​投稿されたメッセージを、じぇーんぶ「ログ」シートに記録してくれる。
​
​■ ⚠️ ちょっと気をつけて！自動ルールチェック機能
​以下のことをやっちゃうと、自動で「閲覧のみ」の権限に変更されちゃうから注意してね
​管理者の人以外がTO ALLを使っちゃったとき
​管理者の人以外が、Chatworkの絵文字を50個以上まとめて投稿しちゃったとき
​使い方がわからなくなったり、何かおかしいなと思ったら、

っていえば強制呼び出しされるよ
